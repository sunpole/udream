# ✦ udream — сонник по Библии

> Христианский сонник: поиск символов, библейские толкования, адаптивный интерфейс / Christian dream symbols search.

**Коротко:** находите христианские толкования снов. Введите слово — получите описание, библейский стих и теги.

## Зачем это нужно

У нас были две книги на английском с толкованиями символов. Искать в них через Ctrl+F — неудобно (ищет куски слов). Вместо этого я сделал простой веб-поиск:

- работает на телефоне и компьютере
- показывает только то, что нужно
- данные живут в браузере, можно добавлять свои символы

Никакого сервера, никакой регистрации. Просто открываете и пользуетесь.

## Возможности

- **Поиск с подсказками** — начинаете печатать, появляются варианты
- **Карточка символа** — кратко или подробно, с библейской ссылкой
- **Тёмная / светлая тема** (в плане — реализуется)
- **Теги** — можно группировать (животные, стихии, служение…)
- **Адаптивность** — не страшно, если у вас старый телефон или большой монитор
